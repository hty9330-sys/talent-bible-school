1. Supabase 프로젝트를 만든다.
2. SQL Editor에서 supabase/schema.sql 전체를 실행한다.
3. Authentication > Users에서 선생님 계정을 만든다.
4. Auth user id와 같은 id로 public.users에 프로필을 추가한다.
   예:
   insert into public.users (id, name, email, role)
   values ('AUTH_USER_ID', '정선생', 'teacher@example.com', 'admin');
5. 앱 설정 화면에 Project URL과 anon public key를 저장한다.
6. 로그인 후 학생 등록, 달란트 지급, 메모 작성, 성경 학습 완료를 테스트한다.
